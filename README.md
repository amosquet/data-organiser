I had too many files, I keep telling my self that I'll sort them but I always come back with a file that hasn't been sorted.
This will hopefully solve that problem.

This program will use regex to automatically sort through an unsorted directory and move it all to a sorted directory.
If regex fails for some reason, or because you think an "AI" is better, you can also do that.
The default is to use regex for filename and content matching to avoid needing the cloud, with fallback to Gemini or a local model through Ollama if the user desires this.
